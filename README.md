# GhostyMusicY

GhostyMusicY es un reproductor de música avanzado para Android con Material Design 3. 

> **Nota**: Esta aplicación está actualmente en una refactorización mayor y las características están siendo añadidas. 

## Características (Próximamente)
- Temas Dinámicos
- Búsqueda Avanzada con filtros 
- Descarga y reproducción offline
- Integración con base de datos propia
- ¡Y mucho más!
